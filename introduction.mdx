## Arquitetura

### Gateaway

Recebe as requisições `HTTP` dos utilizadores e distribui para os microserviços correspondentes.

```bash
ms-gateaway
```

### Weather

Consulta a API externa `(OpenWeather)` para obter informações meteorológicas.

```bash
ms-weather
```

### Geo Service

Usa o `IP` do utilizador para determinar sua localização geográfica.

```bash
ms-geo-service
```

### Database

Armazena os dados climáticos no base de dados `Supabase`.
```bash
ms-database
```
